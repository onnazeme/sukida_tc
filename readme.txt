■遊戲大要
	玩家要度過文化祭前的一個禮拜多的時間及文化祭那兩天的高中生活。
	說不定會和某個角色變得要好，基本上就只是這樣。

	※一部分事件包含男性穿女僕裝的內容。

	攻略對象角色共4位，結局共8種。

■鍵盤操作
	・空白鍵:　訊息視窗的隱藏或顯示
	・Enter鍵:　繼續(＝滑鼠左鍵)
	・Ctrl鍵/Command鍵:　略過訊息


■滑鼠操作
	・左鍵:　繼續/決定
	・右鍵:　訊息視窗的隱藏或顯示
	・滾輪往上:　顯示訊息紀錄
	・滾輪往下:　繼續

■功能
	・SAVE:　開啟存檔畫面
	・LOAD:　開啟讀檔畫面
	・SKIP:　開始略過訊息 (可在CONFIG畫面更改設定)
	・AUTO:　開始自動播放訊息 (可在CONFIG畫面更改速度)
	・LOG:　顯示訊息紀錄
	・SCREEN:　以全螢幕顯示遊戲畫面
	・MENU:　開啟選單畫面
	・CONFIG:　開啟設定畫面
	・CLOSE:　隱藏訊息視窗
	・TITLE: 返回標題畫面

	(略過訊息的方式，「略過未讀」預設為OFF。
	 如果設為ON，將會無關已讀、未讀，可略過所有文章。)
